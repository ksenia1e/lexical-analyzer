# Лексический анализатор учебного языка
Разбивает исходный код на токены: числа, идентификаторы, операторы, ключевые слова.
Распознает лексические ошибки. Выводит таблицы токенов, констант, идентификаторов.

## Структура проекта
- `tables.h` - статические и динамические таблицы символов, констант, идентификаторов
- `lexical.cpp` - лексический анализ: чтение файла, распознавание токенов, обработка ошибок

## Входные данные
`program.txt` - код программы

## Выходыне данные
| Файл | Описание |
| :--- | :--- |
| `constTable.txt` | Таблица числовых констант |
| `dynamicTable.txt` | Таблица идентификаторов (переменных) |
| `staticTable.txt` | Таблица зарезервированных слов и символов |
| `tokens.txt` | Последовательность токенов: код типа, индекс в таблице |
| `errors.txt` | Список ошибок с номерами строк |

## Ключевые слова
`int`, `if`, `else`, `main`, `return`

## Типы токенов
| Код | Тип | Пример |
| :--- | :--- | :--- |
| 0 | Числовая константа | `123`, `42` |
| 1 | Идентификатор | `x`, `count`, `value2` |
| 2 | Зарезервированный символ/оператор | `+`, `-`, `*`, `=`, `<`, `!=`, `==`, `(`, `)`, `{`, `}`, `;` |
